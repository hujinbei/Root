如何安装expres


第一步：安装express


1.打开终端（cmd），在终端输入命令 “ $ mkdir express”
——创建一个名为“expres”的新文件夹用于存放安装express产生的文件


2.新建文件夹后输入命令 “$ sudo npm install -g express”
——安装express的命令
